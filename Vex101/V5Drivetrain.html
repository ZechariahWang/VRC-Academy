Programming a Clawbot drivetrain in V5 Pro Text.

A drivetrain is one of the most important parts of the robot. It is the skeleton of the robot, and is what allows you to move throughout the game. As a result, programming a drivetrain with robust, effective code is essential.

The "tank drive" style drivetrain on the clawbot is relatively simple to program.

To start off, we need to enclose everything in a while loop. This allows for the code to run forever, as we want this code to always run while we are in driver control.

while (true){ }

Second, we need to declare 2 variables, the left power and the right power. The left power is simply just the values of 2 axis of your choice added together. The right power is the same thing, but with subtraction instead. For my case, I will be using axis 3 and axis 1.

int leftpower = Controller1.Axis3.position(percent) + Controller1.Axis1.position(percent);

int leftpower = Controller1.Axis3.position(percent) + Controller1.Axis1.position(percent);

Lastly, all we need to do is tell the motors to spin the amount we set it to. We will get the left motors to spin the left variable power, and the right motors to spin the right variable power.

leftmotor.spin(forward, leftpower, percent);

rightmotor.spin(forward, rightpower, percent);

You've completed all the code you need to make the clawbot move. Changing the axis above will change the joysticks you use when driving the robot. You can also experiment with other types of drives and see which one works best for you.
